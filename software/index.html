Activity Timer for MacOS

This simple activity timer is the right tool for you. It provides a plain and smoothly integrated timer for your OS X status bar.

CC PAL ~ MIDI Control Change iOS app

CC PAL is a MIDI CC controller which allows you to create up to 16 CC sliders. You can record movements for each slider and replay this in a loop. You can also just change the CC value by moving the slider.

X/Y PAL ~ MIDI X/Y pad controller for iOS

X/Y PAL is your handsome X/Y pad MIDI controller. Control up to 8 parameters per pad. Each parameter can be configured.
